# PWA Calderas

Aplicación PWA para gestión de mantenimiento industrial de salas de calderas.

## Stack tecnológico

- React
- Vite
- TypeScript
- TailwindCSS
- Supabase
- PWA
- GitHub Actions

## Funcionalidades

### Autenticación
- Login seguro
- Roles y permisos
- Recuperación contraseña

### Gestión de activos
- Calderas
- Bombas
- Quemadores
- Válvulas
- Sensores

### Checklists
- Diario
- Semanal
- Mensual
- Firma técnica
- Fotografías

### Órdenes de trabajo
- Creación OT
- Seguimiento
- Cierre técnico
- Prioridades

### Preventivos
- Calendario
- Alertas
- Programación automática

### Incidencias
- Alarmas
- Averías
- Fugas
- Paradas

### Dashboard
- KPIs
- MTBF
- MTTR
- Disponibilidad
